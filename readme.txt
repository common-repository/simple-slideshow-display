Download and install our Simple Slideshow Display now.== Description ==Simple Slideshow Display is very nice and simple plugin. It doesn't have lots stuffs on backend. It's simple and implemented exactly what needs.Global Options:Display Controls: Can Display Controls or Hide them.Autoplay: Can enable or disable auto play.Pagination Speed: Can setup Pagination speed in ms.Loop: Loop Images or just have one way slider transition.Stop On Hover: Can Stop Slider transitions on hover.Transition Duration: Transition Duration of Slider moves.Slider Image Height: Integer value in pixel.Take a look at DEMO - http://www.crayfishcreative.com/developer/== Installation ==This section describes how to install the plugin and get it working.Download "simple-slideshow-display.zip" to your computer.Login WordPress as administratorClick on Plugins --> Add NewOn "Install Plugins" page, click on "Upload"Browse and select "simple_note_widget.zip" and click on "Install Now"You can activate it nowGo to your " Responsive Simple slideshows " Menu and adjust the settings== Screenshots ==1.
